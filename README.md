This is my personal blog that I deploy to GitHub Pages with Jekyll.

## Getting Started

If you are using [Nix](https://nixos.org/), you can easily run this locally.

```sh
nix run github:fzakaria/fzakaria.com
```

### Development

```sh
jekyll serve
```
